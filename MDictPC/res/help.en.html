Quick Start:

• Install library files:
Just put all dictionary files(*.mdx/*.mdd) into doc directory of MDict installation folder. They will be imported at every program startup. If you put dictionary files in other directory, you may use "Library"->"Library manager" then select "Import library" or "Add dictionary" to add them to library.

• Lookup:
Input text in the input panel, then press "Enter" to lookup or select entry from the keyword list.
Tips: just type (key in) words you want to lookup, the input focus will automatically switch to input box.

• Multi-Dictionary Lookup:
To lookup multiple dictionaries at the same time, You need to creat a group in the library manager, then put all those dictionaries that need to be lookuped together in this group and mark the group as "Union".

Keyboard shortcut:

• When input focus is in the input panel:
-Enter: Lookup current input/ select entry in entry list

• When input foucs is in the document window
-Enter: Switch input focus to input panel
-Arrow left/right: Jump to previous/next entry
-Input any alpha character will switch input focus to input panel.

• Hotkeys
-Atl+Ctrl+D: Put MDict window into foreground
-Esc: Minimize MDict window into a taskbar icon

• Zoom in/out
Ctrl+ Mouse wheel up/down can change font size in entry list window and text window
